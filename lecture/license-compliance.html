今回の用途について

RAG Chatbot（RAGChatbotWindow.cs）は Unity Editor 上で動く開発者向けツール。エンドユーザーへ配布するランタイムではないため、Unity Runtime Fee の対象外。ただし組織の収益規模によって Pro ライセンスが必要。

🔴 必須対応（リリース前に完了）
  • Houdini ライセンス確認: Apprentice → Indie/Core/FX へ移行。売上見込みで適切なプランを選択。
  • Gemini を Vertex AI 経由に移行: 現行の Google AI Studio 経由はデータ保護の観点でリスクあり。
  • Unity ライセンス確認: 組織の年間売上・調達額が $10万 USD 超なら Pro ライセンスが必要。
🟡 推奨対応（規模拡大時）
  • Anthropic と DPA 締結: 個人情報・機密情報を API に送る場合は Enterprise プランで契約。
  • Google Cloud DPA 締結: Vertex AI 移行後に GCP の DPA も締結する。
  • Notion Business/Enterprise 移行: Notion に機密データを置く場合は DPA 締結のため Business 以上へ。
  • Qt 商用ライセンス検討: 外部配布製品で PySide6 の LGPL 義務を完全に回避したい場合に有効。

「ソース開示が必要になる」ケース整理

条件発生するか理由
GPL ライセンスのパッケージを使う 発生しない 本システムに GPL パッケージなし
AGPL ライセンスのパッケージを使う 発生しない AGPL パッケージなし
PySide6 を改変せず使う（通常） 発生しない LGPL の動的リンク免除に該当
PySide6 本体を改変して配布する 改変部のみ発生 LGPL v3 の要件
MIT/Apache 2.0 のコードを改変する 発生しない コピーレフトではない
